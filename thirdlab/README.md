# Система управления беспилотными такси
## Описание
Программа моделирует работу парка беспилотных такси в многопотончном режиме. Система состоит из:

-**Такси** - обрабатывает заказы

-**Диспетчера** - распределяет заказы

-**Генератора клиентов** - создает новые заказы

-**Потокобезопасной очереди** - хранит заказы
